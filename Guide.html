This is the instruction page

- If you want to start the game you have to enter your name and click the start button

- Now you will enter the game page Where you will see the obstacles arriving near you

- You will be given a gun where you can shoot the obstricals before they destroy your base

- To aim the obstricals you have to left arrow to move left and right arrow to move right

- To shoot them you can access the space bar to shoot the obstacles

- for each set of obstacles you clear , the next set of obstacles coms faster and faster
